# ChatGPT Learning Prompt Template

Use this prompt when you're stuck on concepts, homework, or need deeper explanations.

---

## Copy and paste this template into ChatGPT:

```
I'm learning backend development by building a Multi-Agent Document Intelligence System (DocuMind AI). I'm working on [SPECIFIC TOPIC/CONCEPT YOU'RE STUCK ON].

**My Project Context:**
- Tech Stack: Python, FastAPI, Google Gemini API, ChromaDB, CrewAI
- Project Structure: FastAPI backend with RAG pipeline and multi-agent system
- Current Stage: [Week 1 / Week 2 / etc.] - [What you're working on]

**What I Need Help With:**
[Describe your specific question or confusion]

**What I've Already Tried/Understood:**
[Briefly mention what you know or what you've attempted]

**My Learning Goal:**
I want to understand [CONCEPT] deeply so I can:
1. Explain it in interviews
2. Implement it correctly in my project
3. Understand why we use this approach

**Please:**
1. Explain the concept in simple terms (I'm a beginner)
2. Give a real-world analogy if possible
3. Show a simple code example
4. Explain how it relates to my project
5. Suggest any resources (docs/videos) for deeper learning

Thank you!
```

## Tips for Using This Prompt:

1. **Be Specific**: Replace [SPECIFIC TOPIC] with exactly what you're confused about
2. **Update Context**: Keep the project context updated as you progress
3. **Ask Follow-ups**: Don't hesitate to ask "Can you explain that part again?" or "Can you give another example?"
4. **Request Examples**: Always ask for code examples related to YOUR project structure
5. **Interview Prep**: Ask "How would I explain this in an interview?" at the end
